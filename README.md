# BleDemo

version 0.1.0

ble 蓝牙连接以及发送数据的演示 Demo，包含 java 版本 和 kotlin 两个版本。主分支为 java 版本，另一个分支为 kotlin 版本。
如果需要修改指定的目标设备的连接特征，直接修改 Config 类即可。

关于蓝牙比较不错的资料收集

1.Android BLE基础操作框架，基于回调，操作简单。包含扫描、多连接、广播包解析、服务读写及通知等功能。
